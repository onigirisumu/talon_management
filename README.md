# Запись на прием

Учебный FastAPI-проект для онлайн-записи граждан на прием.

## Возможности

- выбор услуги;
- форма записи с датой, временем и данными гражданина;
- генерация номера талона;
- страница подтверждения записи;
- проверка статуса по номеру талона;
- PDF-талон с QR-кодом;
- перенос записи на другой свободный слот;
- переключение языка интерфейса RUS/KZ/ENG;
- панель сотрудника со списком записей на день;
- простая админ-панель для услуг, слотов, сотрудников и статусов записей.

## Технологии

- FastAPI;
- MongoDB;
- PyMongo;
- Jinja2;
- Uvicorn.

## Запуск

```bash
python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt
.venv/bin/python manage.py
```

Перед запуском должен работать MongoDB:

```text
mongodb://localhost:27017/
```

Сайт: http://127.0.0.1:8001/

Единый вход и регистрация: http://127.0.0.1:8001/login

Демо-аккаунты:

```text
Админ: admin / Admin123!
Сотрудник: staff / Staff123!
```
